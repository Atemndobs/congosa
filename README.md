A react chat-app with node js server powered by socket.io

## Try it out [congosa](https://congosa.netlify.com/)



## Getting started.

- clone the repository

$ cd congosa

#### Go to server directory and install server

$ cd server

 ##### `npm install`

 #### start sever

##### `node start`


#### Install client :

$ cd ..
##### `npm install` 

#### start client

 ##### `npm run start`
